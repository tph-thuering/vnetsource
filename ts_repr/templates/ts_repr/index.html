The Representative Workflow using the EMOD model builds a baseline simulation for a generic location.

To start a new simulation using the EMOD Representative Workflow, select “New Simulation” from the left menu.

To see all your simulations created using either the Basic or Representative workflows, select “My Simulations” from the left menu.

To see simulations that you or others have made public, select “Public Simulations” from the left menu.

Additional tools can be found by selecting “Utilities” from the left menu.

Select “Results Viewer” to request plotted results of simulation outputs.
