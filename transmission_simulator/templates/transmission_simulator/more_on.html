EMOD

EMOD was developed by The Institute for Disease Modeling (IDM). Publications about EMOD can be found at:

http://idmod.org/publications/topic/malaria

  • EMOD models both humans and vectors as individual “agents”.
  • The EMOD model is designed for both human and vector-centric interventions.
  • Currently, on VecNet, the EMOD workflow has more input and output options, and hence is more detailed to setup and run (compared to OpenMalaria).
  • EMOD simulations run longer on VecNet (than OpenMalaria).
  • EMOD documentation can be found at: http://idmod.org/idmdoc/

OpenMalaria

OpenMalaria: was developed by the Swiss Tropical and Public Health Institute (Swiss TPH). Publications about OpenMalaria can be found at:

https://github.com/SwissTPH/openmalaria/wiki/References

  • OpenMalaria models humans as individual “agents”; vector dynamics are modeled with a system of equations.
  • OpenMalaria model is designed for human-centric interventions.
  • Currently, on VecNet, the OpenMalaria workflow is simpler to setup and run (compared to EMOD).
  • OpenMalaria simulations run quicker on VecNet (than EMOD).
