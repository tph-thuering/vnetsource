Basic Workflow


The Basic Workflow is used to model transmission for a specific geographical location.

Either EMOD {% include "transmission_simulator/more_on_emod.html" %} or OpenMalaria {% include "transmission_simulator/more_on_om.html" %} can be used.

The Basic Workflow allows users to define:

  • Seasonality: daily weather measures (EMOD) or monthly EIR (OpenMalaria)
  • Human population: size and age distribution
  • Vector species and parameters that influence malaria transmission
  • Parasite infection and immunity dynamics in both human and vector populations
  • Scaling factors that enable the baseline simulation to be broadly scaled across a range of values
