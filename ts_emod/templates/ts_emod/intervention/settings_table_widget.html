Setting Name Values Description
Start Day Event Start Day 0 to 2.14748e+009 or number of days in data files, whichever is lower, default is 0 The day of the event within the simulation. For most events, this will be a one-time distribution beginning on this day. But, within the software, this is simply the simulation time at which the event becomes active and is handled by the Event Coordinator.
Number Reps Number of Repetitions -1 to 1000, default is -1 (disabled) [Optional] Number of times the event will be repeated at intervals after the first time. It is used for repeated spraying interventions or drug screen-and-treat interventions. This parameter is used in conjunction with Timesteps_Between_Repetitions. A value of -1 disables this optional parameter.
Time Between Reps Time Steps Between Repetitions -1 to 10000, default is -1 (disabled) [Optional] Specifies the repetition interval. This parameter is used for repeated spraying interventions or drug screen-and-treat interventions. This parameter is used in conjunction with Number_Repetitions. A value of -1 disables this optional parameter.
Number Distributions Max Number of Distributions -1 to 2.14748e+009, default is -1 (no limit) Maximum number of interventions to distribute. By convention, no limit is specified by -1.
Demographic Coverage Demographic Usage 0% to 100%, default is 100% Specifies the probability that an individual in the specified Target_Demographic will receive the intervention.
