VecNet Results Viewer

This tool is provided as a portal to the results of Transmission Simulator executions. All of your private executions as well as public executions are available for visualization. Get started by reading and choosing one of the options below!

Scenario-Based

The scenario-based visualization interface allows the graphing and download of data series from a specific Transmission Simulator run. The Scenario-based interface is useful for viewing and comparing data from within the same Transmission Simulator scenario or run. Use the buttons below to browse either EMOD or OpenMalaria results data.
EMOD Results OpenMalaria Results

Custom

The custom interface is provided to visualize data from multiple Transmission Simulator runs on the same chart. This method opens up the output data to create custom charts that are not bound by simulation model, scenario, or homogenous output types.
