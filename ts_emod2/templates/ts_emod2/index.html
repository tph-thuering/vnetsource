Basic Workflow – EMOD

{% endcomment %} The Basic Workflow using the EMOD {% include "transmission_simulator/more_on_emod.html" %} model enables the user to build a baseline scenario for their chosen study location. Users can then add interventions to the baseline including vector control, anti-malarial drugs or vaccines to study the impacts on transmission.

To start a new simulation using the EMOD Basic Workflow, select “New Simulation” from the left menu.

To see all your EMOD simulation created using either the Basic or Representative workflows, select “My Simulations” from the left menu.

To see simulations that you or others have made public, select “Public Simulations” from the left menu.

Select “Utilities” from the left menu for additional tools for working with the EMOD model.
